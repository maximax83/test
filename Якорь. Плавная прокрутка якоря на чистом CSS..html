Что такое якорь?

Якорь - это ссылка, которая ведёт к какому-нибудь месту внутри страницы.
Для якоря нужна ссылка, при нажатии на которую произойдёт перемещение, и элемент с id, к которому произойдёт премещение.
Элементу, до которого будет происходить перемещение, назначается произвольный id. Например, <p id="anchor">.
У ссылки в значении атрибута href указывается # и значение id элемента, до которого будет происходить перемещение. Например, <a href="#anchor">.

Плавная прокрутка на чистом CSS

На верх

Плавную прокрутку якоря на чистом CSS можно реализовать с помощью html {scroll-behavior: smooth;}. Не работает в IE. Не корректный результат с фиксированной шапкой.
